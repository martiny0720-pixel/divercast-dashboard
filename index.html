📖 Gist 생성 가이드

1. GitHub Gist 생성

  • https://gist.github.com 접속
  • "Create new gist" 클릭
  • 4개 파일 추가:
    • clients.json: {"clients": {}}
    • queue.json: {"queue": []}
    • posts.json: {"posts": []}
    • prompts.json: {"prompts": {}}
  • "Create public gist" 클릭
  • URL에서 ID 복사

2. GitHub Token 발급

  • https://github.com/settings/tokens 접속
  • "Generate new token (classic)" 클릭
  • gist 권한 체크
  • "Generate token" 클릭
  • 토큰 복사 (다시 볼 수 없으니 주의!)
